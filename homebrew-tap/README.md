# Homebrew Tap for Deployzy

## Install

```bash
brew tap deployzy/tap https://github.com/jams24/deployzy
brew install deployzy/tap/serverme
```

Or in one command:

```bash
brew install deployzy/tap/serverme
```

## Usage

```bash
serverme http 3000
serverme tcp 5432
serverme tls 443
```

## Update

```bash
brew upgrade deployzy/tap/serverme
```

## Uninstall

```bash
brew uninstall serverme
brew untap deployzy/tap
```
